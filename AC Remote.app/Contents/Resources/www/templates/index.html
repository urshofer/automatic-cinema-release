The Software

Automatic Cinema is a project by Urs Hofer. It consists of a database, a playback control server, various clients to show the media, this remote control app.

Geometrical Storytelling

Automatic Cinema uses a algorithms to decide which media fits the best after the currently playing This technique is called «Geometrical Storytelling».

Dynamic Datastructure

To organize your assets, you can define as many semantical dimensions. Each dimension contains it's own logic where Keywords are linked not only in a hypertextual way, but more in a linguistic. Keywords can relate, oppose or depict a consequence.

Multi Channel

Automatic Cinema can control as many channels as you like. Text, Audio or Video. Channels can be set to master, where they follow their own timeline. Set to slave, they follow the masters rhythm.
